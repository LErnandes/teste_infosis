# No frontend

## Para instalar os pacotes:
```
npm install
```

### Para iniciar:
```
ng serve
```

# No backend

## Para instalar os pacotes:
```
yarn install
```

### Para iniciar:
```
yarn start
```

### Para testar (deve-se testar com o backend iniciado):
Em um terminal:
```
yarn start
```

Em outro terminal:
```
yarn test
```
